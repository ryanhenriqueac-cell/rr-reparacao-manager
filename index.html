FAQ

Perguntas rapidas.

O sistema ja esta pronto para entrar?

Sim. O botao Entrar abre o dashboard; se o usuario nao estiver logado, o proprio sistema mostra a tela de login.

A landing fica no mesmo projeto?

Sim. O dominio abre a landing em index.html e o sistema continua nas paginas internas, como dashboard.html.

Da para usar em celular?

Sim. A interface da landing e do sistema foi ajustada para desktop e celular.

Como solicito acesso?

O botao Solicitar acesso abre o cadastro para o usuario pedir liberacao no RR Manager.
